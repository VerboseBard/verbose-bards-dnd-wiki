# Verbose Bard's D&D Wikipedia

This repository publishes the static web version of Verbose Bard's D&D Wikipedia.

The source pages live in `wiki/` as Markdown. The static HTML site is generated by `tools/build-wiki-site.ps1` into `wiki-site/`.

## Local Build

Run this from the repository root:

```powershell
powershell -ExecutionPolicy Bypass -File tools\build-wiki-site.ps1
```

Then open:

```text
wiki-site\index.html
```

## Publishing

GitHub Actions builds and deploys the site to GitHub Pages whenever changes are pushed to `main`.

The workflow:

- builds `wiki-site/`
- adds `.nojekyll`
- uploads the generated site as a Pages artifact
- deploys it to GitHub Pages

## What Is Published

Only the public wiki source and generated static site are intended for this repository. Raw campaign transcripts, local scratch files, temporary exports, and private workspace notes should stay outside this project unless intentionally promoted into the public wiki.
