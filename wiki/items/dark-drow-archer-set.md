# Dark Drow Archer Set

The Dark Drow Archer Set is a three-piece dark elf relic set associated with silent hunters of the drow. The set centers on shadow-marked archery, generated ammunition, web-binding, and predatory stealth.

## Set Pieces

- [Dark Drow Archer's Bow](dark-drow-archers-bow.md): part 1 of 3.
- [Webbed Quiver](webbed-quiver.md): part 2 of 3.
- [Purple Dragon Archer Gloves](purple-dragon-archer-gloves.md): part 3 of 3.

## Set Identity

The bow provides the killing shot and shadow mark. The quiver generates arrows and specialty ammunition. The gloves stabilize the set's magic, protect the wielder, and provide extradimensional storage.

Together, the pieces form a stealth archer relic suite built for pursuit, ambush, and battlefield control rather than simple damage.

## Campaign Notes

Session 8's transcript flagged the archer set as an unfinished item-card area. The later supplied cards define the three known pieces as the Dark Drow Archer Set.
