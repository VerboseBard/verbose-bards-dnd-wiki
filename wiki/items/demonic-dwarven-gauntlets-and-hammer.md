# Demonic Dwarven Gauntlets and Hammer

The Demonic Dwarven Gauntlets and Hammer are an artifact-level dwarven set recovered from [Wendell](../people/wendell.md)'s corrupted dwarven servants.

## Description

The gauntlets match the demonic dwarven glove previously carried by the [Moon Stone Collectors](../factions/moon-stone-collectors.md), though the recovered set has slight variations.

The hammer is a large two-handed weapon covered in runes and symbols that interact with the gauntlets. The set appears extremely powerful, but it is demonic and may carry serious corruption risks.

Later discoveries connect these artifacts to the same dwarven demonic tradition that produced the [Demonic Dwarven Goo](../concepts/demonic-dwarven-goo.md), including containment or discharge mechanisms for the infectious substance.

## Portal Function

The set can open a hidden route from the [Dwarven Egg Fortress](../places/dwarven-egg-fortress.md) to [Wendell's Sixth-Layer Fortress](../places/wendells-sixth-layer-fortress.md). Opening that route requires someone to attune to and wear the gauntlets.
