# Dwarven Mastercraft Tools

The Dwarven Mastercraft Tools are an ancient magical toolset found in a hidden worksite on the thawing road north of [Frostfire Keep](../places/frostfire-keep.md).

## Description

The tools are stored in a light magical box. Individual tools are miniaturized while stored, then resize to proper working scale when removed.

The set includes smithing, woodworking, jeweler's, and related crafting tools. Work performed with the set gains exceptional precision and reliability.

In [Moon Stone Collectors, Part 14](../sessions/moon-stone-collectors-part-14.md), [Paul](../people/paul.md) considers using jeweler's or rune-carving tools to practice modifying ancient dwarven light crystals into radiant anti-goo devices.

## Discovery

In [Moon Stone Collectors, Part 12](../sessions/moon-stone-collectors-part-12.md), the party finds the tools after helping a winter wolf pack recover pups trapped behind an ancient warded doorway.
