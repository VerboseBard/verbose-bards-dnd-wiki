# Enchanting Dagger

The Enchanting Dagger is a magical blade recovered from [Lyris the Deceiver](../people/lyris-the-deceiver.md).

## Description

The dagger is tied to enchantment or deception magic. Its exact properties still require identification.

In [Moon Stone Collectors, Part 9](../sessions/moon-stone-collectors-part-9.md), the dagger's memory is used as the buy-in for the high-stakes game at [Ebonwing Manor](../places/ebonwing-manor.md).

In the reconstructed [Moon Stone Collectors, Part 10](../sessions/moon-stone-collectors-part-10.md), [Juan](../people/juan.md) attempts to use the dagger against [June](../people/june.md) while still compromised by enemy influence, but the attack fails.
