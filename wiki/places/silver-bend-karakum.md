# Silverbend Source Note (Karakum)

This page is retained as the [Fifth Age](../concepts/fifth-age.md) source note for the [Karakum](karakum.md) campaign material. The canonical location page is [Silverbend](silver-bend.md).

## Fifth Age

The city is described as a satyr settlement with flowers everywhere, two temples, a large library, and a bazaar. It functions as an important northern anchor for the [Karakum](karakum.md) trade and guild network.
